1. Выкачайте репозиторий hillel-javaee себе локально
2. Создайте там бранч, переключитесь в него
3. Создайте в нем папку lab1/<ваша фамилия кириллицей в нижнем регистре>
4. Напишите программу, которая выводит на экран числа от 1 до 100. При этом вместо чисел, кратных трем, программа должна выводить слово «Fizz», а вместо чисел, кратных пяти — слово «Buzz». Если число кратно и 3, и 5, то программа должна выводить слово «FizzBuzz»
5. Закоммитить в новую ветку
6. Создать pull request через github.com, добавить меня в reviewers
